SnagOnline
==========

Online version of snag that uses NodeJS, Express Web Server, and Pomelo Server Framework.


XML Level Format
================
<tt>
<maps>
	<map name="map1" width="1000" height="1000">
		<hooks>
			<hook x="100" y="100"></hook>
		</hooks>
		<walls>
			<wall x= "100" y="100" height="200" width="40"></wall>
		</walls>
	</map>
</maps>
</tt>
=================
End
